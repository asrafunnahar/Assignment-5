1️⃣ What is the difference between var, let, and const?

Answer: var vs let vs const: var is function-scoped and can be leaked out of blocks. let and const are block-scoped (locked to {}). Use const by default, let if the value changes, and avoid var.

2️⃣ What is the spread operator (...)?

Answer: Unpacks an array or object into individual elements or key-value pairs (e.g., [...arr] makes a quick copy).

3️⃣ What is the difference between map(), filter(), and forEach()?

Answer: map()transforms every item and returns a new array of the same length. filter() returns a new array containing only items that match a condition. forEach() just loops to perform actions and returns nothing.

4️⃣ What is an arrow function?

Answer: A arrow function syntax that does not create its own this context, instead inheriting this from its surrounding code.

5️⃣ What are template literals?

Answer:Strings written with backticks that let you insert variables directly using ${variable} and easily write strings across multiple lines.
